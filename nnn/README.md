# NNN

[link](https://github.com/jarun/nnn)

## Install
```bash
brew install nnn
```

## Configuration

* Show icons: a) git clone; b) compile with O_NERD=1; c) replace nnn at /bin
* Create variables at oh-my-zsh config file

Configure settings at .zshrc

## Keymap

````sh
e                               edit file with lvim
b                               show bookmarks
;                               show plugins
``````
